## Контекст продукта

**VIRAL_MPV** (Viral Monitor) — сервис для creator-ов: мониторит вирусные рилсы конкурентов, разбирает их (скачивание → транскрипция → анализ кадров) и помогает создавать свои аналоги через AI-студию.

Текущее меню навигации: **Монитор · Разбор · AI-студия · Аналитика · Хэштеги · ИИ-агент**.

Pipeline разбора уже сделан (Piratex-style, 3 шага: Скачивание → Транскрипция → Анализ кадров). В этой задаче — переделываем **результат разбора** и **AI-студию**.

## Дизайн-система (используй ровно это)

```css
:root {
  --bg0: #000;          /* page background */
  --bg1: #0a0a0a;       /* primary surface */
  --bg2: #141414;       /* elevated surface */
  --bg3: #1c1c1c;       /* hover/active surface */
  --br:  rgba(255,255,255,.06);  /* subtle borders */
  --br2: rgba(255,255,255,.12);  /* visible borders */
  --t1:  rgba(255,255,255,.92);  /* primary text */
  --t2:  rgba(255,255,255,.7);   /* secondary text */
  --t3:  rgba(255,255,255,.45);  /* tertiary/labels */
  --t4:  rgba(255,255,255,.3);   /* muted/captions */
  --teal: #14b8a6;
  --teal-hi: #2dd4bf;
  --serif: 'Playfair Display', 'EB Garamond', Georgia, serif;
  --sans: ui-sans-serif, system-ui, -apple-system, 'Segoe UI', Roboto, sans-serif;
  --mono: ui-monospace, 'SF Mono', Menlo, Consolas, monospace;
}
body { background: var(--bg0); color: var(--t1); font-family: var(--sans); }
```

Тоны:
- Заголовки H1/H2 — **serif** (Playfair Display), белый, large, weight 400, letter-spacing -.01em
- Метки/labels/времени/счётчики — **mono**, uppercase, letter-spacing .06–.1em, t3/t4
- Body — sans-serif, 13–14px, t1/t2

Бейджи платформ:
- INSTAGRAM: `#833ab4` фон, белый текст, 9.5px mono, padding 2/7, radius 4
- YOUTUBE: `#c00`
- TIKTOK: `#111` с белым бордером

Кнопки:
- Primary: фон `var(--teal)`, текст `#000`, weight 600, padding 9/18, radius 9, hover translateY(-1px) + teal box-shadow
- Ghost: transparent, border `var(--br2)`, текст `var(--t2)`, тот же размер
- Pill button (фильтры): bg2 фон, br border, textsm, padding 6/12, radius 8

```
AI-студия                                        [+ Создать с нуля]
```

### Tab-bar (5 вкладок)

- Вкладки — pill-стиль, активная: bg3 фон + teal текст-shadow
- Цифры в круглых счётчиках mono 10px
- "Загрузка" — НОВАЯ вкладка, её и проектируем как основной экран

### Сетка карточек (вкладка "Загрузка")
4 колонки на десктопе, 2 на планшете, 1 на мобилке. Карточка вертикальная (9:16 thumbnail сверху + meta снизу):

```
┌─────────────────────┐
│ [INSTAGRAM]   [♥][×]│  ← бейдж платформы слева, актонки справа
│                     │
│   thumbnail 9:16    │
│                     │
│              1:16   │  ← duration в правом нижнем углу
│  Готово             │  ← статус-таблетка teal в левом нижнем
├─────────────────────┤
│ Если вы внедрите    │  ← title 13px t1, 2 строки max
│ эту простую...      │
│ alexeylinetsky      │  ← author mono 11px t3
│                     │
│ ♥ 11.7K  💬 59      │  ← stats mono 10.5px t3
│ ER 0.5%  ⚡ 4.7×    │  ← engagement & velocity
│                     │
│ [Скопировать] [✓]   │  ← кнопка primary + bookmark icon
└─────────────────────┘
```

**Важно:** сделай 8-12 mock-карточек разных аспектов (разные thumbnail-цвета через linear-gradient, разные авторы, разные статусы).

### Состояния карточек
- **Готово** (default): teal-таблетка, кнопка "Скопировать"
- **Лайкнутая**: ♥ заполнено teal в правом верхнем
- **Hover-карточки**: border теплеет до `--br2`, легкий lift `transform:translateY(-2px)`, slow .2s ease, shadow

### Логика клика "Скопировать"
1. Карточка анимированно "выпрыгивает" вверх (.4s)
2. Tab-bar переключается на "В обработке" с animated indicator
3. Видео появляется в "В обработке" с pipeline-индикатором (тонкий progress bar внизу карточки)
4. Pipeline шуршит и стримит варианты:
   - Заголовок (3 варианта)
   - Сценарий (структура: hook → main → CTA)
   - Хэштеги
   - Описание

Покажи это переключение работающим — клик по любой "Скопировать" → tab switch + новая карточка в обработке. Всё на одной странице, без роутинга.

### Вкладка "В обработке" (нужно показать тоже)
Те же карточки, но:
- Поверх thumbnail полупрозрачный teal-overlay с пульсирующей точкой
- Под title — мини-pipeline (3 dots): ⬇ ✓  📝 в работе  ✦ pending
- Кнопки заменены на: [Открыть] [Отменить]

## Экран 2 — Результат "Разбор" (нижняя часть страницы)

После завершения pipeline (3 шага сверху уже показал статус "Готово"), под ними должны быть две большие секции.

### Секция А — Покадровый разбор

**Заголовок секции:**
```
─── ПОКАДРОВЫЙ РАЗБОР · 18 ───
```
mono 11px uppercase letter-spacing .12em t3, центрирован, с тонкими горизонтальными линиями слева и справа.

**Тайминг кадров** (важная деталь):
- Первый кадр: `00:00` — длится **3 секунды**
- Все остальные: каждые **4 секунды**
- Тайминги: `00:00` `00:03` `00:07` `00:11` `00:15` `00:19` `00:23` `00:27` `00:31` `00:35`...
- Для видео 1:16 (76 секунд) ≈ 18 кадров

**Сетка карточек:** 4 колонки на десктопе, gap 16px, card aspect ratio 9:16 для thumbnail части.

**Анатомия карточки кадра:**
```
┌───────────────────┐
│ 00:03  [тег]      │  ← top: timestamp pill (mono, dark bg) + scene tag
│                   │
│   thumbnail       │  ← фото-кадр или fallback gradient (9:16)
│   9:16 ratio      │
│                   │
│   ПРИВЫЧКИ        │  ← если есть текст на видео — overlay поверх
│   НОМЕР ОДИН      │
├───────────────────┤
│ ТЕКСТ НА ЭКРАНЕ   │  ← mono uppercase 9.5px t4 letter-spacing .08em
│ Привычки номер... │  ← regular 13px t1, weight 500, line-height 1.4
│ (HABITS NUMBER 1) │  ← перевод курсивом 11.5px t3
│                   │
│ ВИЗУАЛ            │  ← mono uppercase t4
│ Молодой человек   │  ← 12.5px t2 line-height 1.55, 3-5 строк
│ в чёрной одежде   │
│ сидит за столом.. │
│                   │
│ микрофон · стол   │  ← теги объектов mono 10px pill mini
│ · интерьер        │
└───────────────────┘
```

**Цвета scene-тегов:**
- `Говорящая голова` — голубой (#3b82f6, semi-transparent fill)
- `Текстовая карточка` — янтарный (#eab308)
- `Перебивка` — серый (var(--t3))
- `B-roll` — фиолетовый (#a855f7)
- `Анимация` — teal (var(--teal))

**Hover на карточке:** легкий lift, border теплеет, тег сцены немного увеличивается (scale 1.05).

**Mock-данные:** 12 карточек разных типов с реалистичным русским текстом про "Привычки миллионеров" (см. скриншот Piratex выше — там есть готовые тексты для копирования).

## Контекст-референсы (для тебя, Claude Design)

**Piratex.ai** — главный визуальный референс. Стиль:
- Чисто чёрный фон
- Teal акценты (#14b8a6 ish)
- Serif-заголовки контрастируют с mono-метками
- Карточки на pure black с минимальными бордерами
- Statuses в pill-форме всегда

**Что НЕ делаем (anti-patterns):**
- Никаких purple gradients
- Никаких generic Inter/Roboto
- Никаких rounded card-ов с большим radius (>12px) — выглядит casual
- Никаких ярких цветных кнопок кроме teal primary
